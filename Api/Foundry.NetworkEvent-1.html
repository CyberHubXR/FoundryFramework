Class NetworkEvent<T>

Namespace
Foundry
Assembly
Foundry.Core.dll

This event is synced across the network. In most ways it functions identically to UnityEvents, but it is even less performant because of the serialization and network overhead. Use only for events that need to be synced across the network.
